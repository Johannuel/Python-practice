# Visual Assets Generation

I do not generate binary image files (JPG, PNG). Instead, I create high-quality, scalable visual components using web-native technologies:

## Supported Techniques
- **Vanilla CSS**: Used for shapes, gradients, and layout-based visual structures.
- **SVG**: Used for complex, scalable vector graphics, icons, and diagrams.
- **HTML Canvas**: Used for procedural graphics or data visualizations.

## Design Strategy
1. **Code-First**: All visuals are defined in code.
2. **Scalability**: SVG is preferred for resolution-independent graphics.
3. **Consistency**: Use CSS variables for color palettes and sizing to ensure brand consistency across components.
